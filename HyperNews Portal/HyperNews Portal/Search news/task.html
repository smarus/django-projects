Description

You kept adding news to your site and everything was great. At some point, there was a lot of news materials on your site and it became difficult to find what you need. You decided to add a search to your site.

Objectives

Add a search form to your main page with one input element with the name q . The form should send GET requests to the same /news/ page. The format of the result page stays the same but includes only that specific piece of news whose title contains the input data.

Do you remember the "Coming soon" page? Now your site is complete and all requests to that page should be redirected to the main page at localhost:8000/news/.

Great job!

Extra

This step is NOT part of the assessment but for ones who eager to prettify the result with the CSS. If you follow all the requirements related to page layouts, then you can add the base.css file in your project.

Create the path static/css in the root of your project and put the base.css file there. Then paste at the top of each template you've created:

{% load static %}
<link rel="stylesheet" target="_blank" href="{% static 'css/base.css' %}">

Tweak your settings a little bit:

DEBUG = True
 
STATIC_URL = "/static/"
STATICFILES_DIRS = [os.path.join(BASE_DIR, "static")]

And add an URL pattern to match static content:

from django.conf import settings
from django.conf.urls.static import static
  
urlpatterns += static(settings.STATIC_URL)

Launch your application again, and look at the result!

You can also play around with the content of the static/css/base.css file and see what you get else.
